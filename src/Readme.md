# Basic String & Number Problems

This repository contains solutions to commonly asked **coding interview problems** focused on strings and numbers.  
Each problem is implemented **with and without using inbuilt methods** to strengthen core logic skills.

---

## 📌 Problems Covered

### 1. Reverse a String
- Reverse a given string
- Implementations:
    - Using inbuilt methods
    - Without using inbuilt methods (manual iteration)

---

### 2. Reverse an Integer
- Reverse a given integer number
- Implementations:
    - Using inbuilt methods
    - Without using inbuilt methods (mathematical approach)

---

### 3. questions.Palindrome Check
- Determine whether a string is a palindrome
- Approaches:
    - By reversing the string and comparing
    - Using two-pointer technique (without extra space)
